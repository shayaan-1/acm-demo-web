# 📝 Full Stack Todo App

## 🚀 Quick Start Guide (5-Minute Setup)

### 🧩 Prerequisites
- **Node.js** (v14 or higher)
- **MongoDB** (local or cloud via MongoDB Atlas)

---

## 🖥️ Setup

```bash
git clone <url_of_repo>
cd frontend
npm i
cd ..
cd backend
npm i
npm run dev (in both folders frontend and backend)
